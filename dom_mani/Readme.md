# This is a sample project to learn the best practices in developing a node app
  * To create a new package.json file use 
    * npm init
  * You can set several config options for the init command 
    * npm set init.author.name 'sowju'
  * To specify the packages the project depends on, you need to list the packages you like to use in package.json in dependencies 
  * to install a package and add it to the package.json use
    * npm install mocha --save
  * To run the server file app.js type the following command
    * node app.js
